# HealthMatebot
**HealthMate Bot** — Telegram-бот для отслеживания здоровья и образа жизни.

## Описание
Бот позволяет:
- Записывать количество выпитой воды (/water)
- Отслеживать часы сна (/sleep)
- Фиксировать время физических упражнений (/exercise)
- Получать персональные советы по улучшению здоровья (/advice)
- Просматривать список команд и подсказки (/help)

Бот хранит данные пользователя и работает стабильно.

## Команды
| Команда | Описание |
|---------|----------|
| /start  | Запуск бота и приветствие |
| /help   | Список всех команд |
| /water [мл] | Добавить количество выпитой воды |
| /sleep [часы] | Добавить количество сна |
| /exercise [мин] | Добавить время упражнений |
| /advice | Получить персональные советы |

## Используемые технологии
- Python 3.14+
- [Aiogram 3.x](https://docs.aiogram.dev/en/dev/)
- SQLite/JSON для хранения данных
- Асинхронное программирование (async/await)
